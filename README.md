# AI Learning Arena

AI Learning Arena is an interactive AI-powered platform designed to simulate real-world backend engineering and problem-solving scenarios.

## Features
- FastAPI backend APIs
- Interactive frontend UI
- Scenario-based AI workflow
- Middleware and CORS integration
- Real-time API communication

## Tech Stack
- Python
- FastAPI
- HTML
- CSS
- JavaScript

## Project Structure
- Backend API using FastAPI
- Frontend interface using HTML/CSS/JS
- Rule engine and orchestrator modules
